Secure Message Service with End-to-End Encryption and Smartcard Support

  • Description
  • Code

Description

Python-based message service featuring end-to-end encryption.

In this message service, all users authenticate themselves using their Portuguese Citizen Card. This service uses ECDHE for key exchanges, AES for message ciphering and the PSS signature scheme.

The system as a whole has been designed to enable the following features:

  • Message authentication, confidentiality, and integrity.
  • Forward and backward secrecy.
  • Identity preservation.
  • Destination validation.
  • Support for cipherspec negotiation.
  • Conversation and participant consistency.
  • Implementation of an information flow policy (Bell-LaPadula).
