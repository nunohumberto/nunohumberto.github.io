Description

This web application allows users to play several titles which belong to retro consoles.

At the moment, the following consoles are supported:

  • NES
  • Gameboy
  • Gameboy Color
  • Gameboy Advance

The platform automatically detects uploaded ROMs and crawls Nintendo Wikia for title information.
