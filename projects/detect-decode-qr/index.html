Detecting and Decoding QR Codes using OpenCV

  • Description
  • Code

Description

My first contact with OpenCV.

This tool captures frames from a locally connected webcam and proceeds to detect and decode any present QR code.

Although 40 different versions of QR codes currently exist, this application focuses in decoding one of the most common versions (25x25). All mask patterns are supported.
